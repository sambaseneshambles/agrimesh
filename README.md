# AgriMesh

AgriMesh is a decentralized agritech platform designed to deliver real-time soil, weather, and crop data to rural farmers in Africa using LoRa mesh networks. By operating independently of cellular or broadband infrastructure, AgriMesh bridges the digital divide for smallholder farmers, helping them reduce crop loss and optimize resource usage.
